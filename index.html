Omniscient RPC for Windows

Omniscient is a simple program that takes the currently active window's title and sets it as a Rich Presence on Discord.

Custom icons list

Firefox (Nightly and Dev too), Chrome, Edge, Internet Explorer, Opera, Paint, paint.net, Inkscape, Premiere Pro, After Effects, Vegas Pro, VLC, Unity 20XX, Visual Studio, Notepad++, Steam, Discord and Skype
